# IntrAgent.github.io

Website repository for **IntrAgent: An LLM Agent for Content-Grounded Information Retrieval through Literature Review**.

This repository hosts the project page prepared for **ACL 2026**, including the paper overview, method summary, benchmark details, experimental results, and resource links.

## Project Links

- Project website: `index.html`
- Main code repository: https://github.com/FengboMa/IntrAgent
- IntraBench dataset: https://huggingface.co/datasets/IntrAgent/IntraBench
